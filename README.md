# AutoAskAI# AutoAskAI

A Python automation script that batch-processes knowledge points through a custom teaching prompt using the OpenAI API.

## Why I Built This / 為什麼做這個
I wrote a super strong teaching prompt that makes AI explain concepts from scratch. I wanted to run hundreds of knowledge points through it and compile the results into a textbook. Doing it manually one by one was too slow, so I wrote this script to automate the entire process — what took hours now takes seconds.

## Files / 文件
- AutoAskAI.py — The automation script / 自動化主程式
- super_prompt.txt — The custom teaching instruction template / 超強教學指令模板

## Built With / 使用技術
- Python
- OpenAI API
